Details

In-depth details

Crystal Client requires FabricMC & Fabric-Api to run.

  1. Download and install Fabric Loader (click here)
    Install fabric loader for whatever version you want to play on
  2. Download and install Fabric-Api (click here)
    Copy fabric-api mod into %appdata%/.minecraft/mods
  3. Download and install Crystal Client (click here)
    Copy Crystal Client mod into %appdata%/.minecraft/mods
